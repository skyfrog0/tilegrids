# tilegrids

瓦片金字塔浏览器

## 瓦片金字塔可视化页面

- 实现左右双面板瓦片金字塔网格可视化
- **左侧 WebMercator (EPSG:3857)**：红色网格，0级1张，每级×4（z级共 2^z × 2^z 张）
- **右侧 WGS84 经纬度直投 (EPSG:4326)**：蓝色网格，0级2张(2×1)，每级×4（z级共 2^(z+1) × 2^z 张）
- 交互：鼠标拖拽平移、滚轮缩放（每次层级±1，视觉倍率2），以鼠标位置为缩放中心
- 联动：两边 zoom 同步；中心点通过 Mercator↔经纬度 互转保持地理位置一致
- 技术要点：
  - 每个瓦片屏幕固定 256px（TILE\_PX），层级变化时瓦片数量翻倍，自然实现2倍缩放
  - WEBMERCATOR\_R = 20037508.34 = R\_earth \* π，公式：y = R/π \* ln(tan(π/4 + latRad/2))
